# The Dactyl Keyboard
The Dactyl is a parameterized, split-hand, concave, columnar, ergonomic keyboard.
The project is a fork of [kennykaye](https://github.com/kennykaye/dactyl-keyboard) which has the awesome modified palm-rest.
